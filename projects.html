Ace Script

AceScript is a full-stack web app developed as my bootcamp final project. Designed to help junior developers improve their skills, it lets users solve coding challenges directly in the browser on a code editor and get instant feedback via pre-written test cases and console logs created by the user. Built with Next.js, Tailwind CSS and MongoDB, it features user authentication, challenge filtering, and a user profile to track progress. This was a group project over several weeks from inception to completion and showcases my ability to work and deliver on something as part of a team
